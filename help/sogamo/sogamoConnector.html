API()method
public function API($apiKey:String, $player_id:String, $username:String, $firstname:String, $lastname:String, $dob:String, $gender:String, $signup_date:Date = null, $updated_date:Date = null, $email:String, $relationship_status:String, $no_of_frds:int, $status:String, $credit:int, $currency:String):void

Connects your game to the Sogamo API, this interacts with sogamoSystem calling the same method on sogamoSystem class, you can start to call methods at any moment even before connection to sogamo system in case is needed.

Parameters

$apiKey:String — The API key for your game. This can be found in Sogamo dashboard.
 
$player_id:String — The player’s player ID. For Facebook games, we recommend that you use the player’s Facebook ID. Otherwise, this can be the player ID that you assign to the player.
 
$username:String — The player’s username. For Facebook games, we recommend that you use the player’s Facebook username. Otherwise, this can be the username that the player chooses, or left null if not applicable.
 
$firstname:String — The player’s first name
 
$lastname:String — The player’s last name
 
$dob:String — The player’s date of birth in the format of MM-DD-YYYY. Example: 02-20-1980
 
$gender:String — The player’s gender, which is either male or female. Example: male
 
$signup_date:Date (default = null) — The date that the player signed up to play your game.
 
$updated_date:Date (default = null) — The date that the player last updated his details. The format is as in the example. For Facebook games, this is the “updated_time” field in the User object.
 
$email:String — The player’s e-mail address
 
$relationship_status:String — The player’s relationship status: Single, In a relationship, Engaged, Married, It's complicated, In an open relationship, Widowed, Separated, Divorced, In a civil union, In a domestic partnership
 
$no_of_frds:int (default = NaN) — The player’s number of friends
 
$status:String — Player’s status: New Old This is so that Sogamo does not mistake an old player as new, because player can be new to our system but have been playing your game for some time.
 
$credit:int (default = NaN) — Player’s Facebook credit balance
 
$currency:String — A comma-separated string containing all of player’s virtual currencies and balances Example: G=1,S=20,B=90
